# VRChat LogMonitor to OSC

VRChat起動中のログを見てリアルタイムのインスタンス人数とインスタンス滞在時間をOSCでVRChatに送るツール。

# Requirement

* Windows 10 以降想定
* ※Windows以外だと多分動きません


## Python

* Python 3.8以上
* python-osc 1.8.0

```bash
pip install python-osc
```
